# Cement-Strength-Polynomial-Regression
Here, we use Polynomial Regression in order to predict the concrete strength based on parameters like cement,ash,slag etc..,

## Installation

### Requiremenmts
* Python 3.3+ or Python 2.7 
* scikit learn
* numpy 
* pandas
* matplotlib

Install the libraries from the pypi.org with the following commands.
```
pip install scikit-learn
pip install numpy
pip install pandas
pip install matplotlib
```
### Please do refer [stackoverflow](https://stackoverflow.com/) for any errors during installation.
